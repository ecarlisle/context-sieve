| CHANGELOG.md | Release team | `@releases` |

## Review Requirements

| Change Type | Required Reviewers |
|---|---|
| New document | Area owner + 1 peer |
| Content change | Area owner |
| Terminology change | Architecture team |
| Cross-area change | Both area owners |
| README.md change | Architecture team + 1 |

## Update Cadence

- **Feature releases:** All affected docs updated in the same PR
- **Patch releases:** Docs updated only if behavior changed
- **Meta documentation:** Updated as needed, no release cadence
