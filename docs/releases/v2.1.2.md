# v2.1.2 — Documentation Completion Pass

**Release date:** 2025-06-25

## What Changed

This release closes all documentation gaps identified in the v2.1 development cycle. It converts accumulated architecture knowledge into a complete, navigable documentation system.

**No runtime code was modified.** This is a documentation-only release.

## Why

As the project grew, documentation coverage became uneven. New features (provider validation, multi-run, replay, extensions) had thorough docs, but foundational areas (timeline model, metrics semantics, investigation workflows) had gaps. This release fills every gap and reorganizes the entry point for navigability.
